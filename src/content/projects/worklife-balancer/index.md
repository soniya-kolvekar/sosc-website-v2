# WorkLife Balancer

**WorkLife Balancer** is a productivity and wellness platform designed to **manage tasks, track mental health, and promote healthy work habits**.  
It combines time management techniques with mood tracking and break reminders to help users maintain a sustainable work-life balance.

The platform provides structured task management with Pomodoro timers, personalized insights, and weekly analytics to optimize productivity while prioritizing mental well-being.

---

## Key Features

- **Task Management** – Create, organize, and track tasks with priority levels
- **Pomodoro Timer** – Structured work sessions with built-in breaks
- **Mood Check-Ins** – Track emotional well-being throughout the day
- **Break Activities** – Guided break suggestions to maintain mental health
- **Weekly Dashboard** – Comprehensive analytics and insights on productivity patterns
- **Daily Streaks** – Gamified progress tracking to build consistent habits
- **Personalized Insights** – Data-driven recommendations for work-life balance
- **Session Management** – Secure user authentication and preferences

---

## Tech Stack

WorkLife Balancer is built using modern web technologies:

- **React & Vite** – Fast and responsive frontend interface
- **Node.js & Express** – RESTful backend APIs
- **MongoDB** – Database for user data and activity tracking
- **JWT Authentication** – Secure user sessions
- **Passport.js** – Authentication middleware
- **Cron Jobs** – Automated weekly data processing
- **Chart Visualization** – Interactive progress and analytics displays

---

## Project Goals

- Help users maintain sustainable productivity levels
- Promote mental health awareness through mood tracking
- Reduce burnout with structured breaks and time management
- Provide actionable insights based on work patterns
- Build healthy habits through consistent tracking and gamification
