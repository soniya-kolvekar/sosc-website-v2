# Resume Parser

**Resume Parser** is an automated recruitment tool designed to **extract, structure, and analyze candidate information from resumes**.  
It converts unstructured resume data into clean, standardized candidate profiles, significantly reducing manual screening effort.

The platform supports resumes in multiple formats and enables recruiters to quickly access relevant candidate details in a consistent and searchable form.

---

## Key Features

- **Automated Resume Parsing** – Extracts candidate information from PDF and document formats
- **Structured Candidate Profiles** – Organizes data into standardized fields for easy review
- **Information Extraction** – Identifies and processes:
  - Personal and contact details
  - Matched skills
- **Fast Screening** – Helps recruiters shortlist candidates efficiently
- **Scalable Design** – Handles large volumes of resumes with consistent accuracy

---

## Tech Stack

Resume Parser is built using modern backend and NLP tools:

- **Next.js** – Frontend interface for reviewing parsed profiles
- **Node.js & Express** – Backend APIs and orchestration
- **Python** – Resume processing and text analysis
- **NLP Techniques** – Entity extraction and data structuring
- **PDF/Text Processing Libraries** – Resume content extraction

---

## Project Goals

- Automate resume screening and data extraction
- Reduce manual effort in recruitment pipelines
- Improve consistency and accuracy of candidate evaluation
- Provide a reusable open-source base for HR tech solutions
